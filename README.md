# Webgl tools WIP[![experimental](http://badges.github.io/stability-badges/dist/experimental.svg)](http://github.com/badges/stability-badges)


## Instalation
```bash
git clone https://github.com/JordanMachado/webgl-tool
npm install
```

## Run it
```bash
npm start
```
